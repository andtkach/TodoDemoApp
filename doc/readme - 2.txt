DemoTodoApp. Part 2.

Add authentication and IAM service using NodeJS Web API service
Add BFF for web app communication with external services using Nginx
Add message processing with AI using processing service and message queue communication to generate voice from the task text.

Build and run complete composition locally for development and debugging
Kill infrastructure in Docker Desktop

Azure deployment
1. Rebuild container images locally [ build-containers.cmd ]
2. Push container images to the remote registry [ publish-containers.cmd ]
3. Restart Azure AppServices
